# Nabila Tabassum | Agricultural Engineering Student Portfolio

A professional, high-performance portfolio website for Nabila Tabassum, an Agricultural Engineering and Technology student at Bangladesh Agricultural University.

## Deployment
Live Website: [ https://tabassumnabila979-web.github.io/My-Portfolio/]( https://tabassumnabila979-web.github.io/My-Portfolio/)

## Features
- Modern UI/UX: Minimalist dark theme inspired by the PlanetScale design system.
- Professional Aesthetics: High-contrast typography, glassmorphism card components, and nature-green accents.
- Animation Suite:
  - Intersection Observer API for scroll-reveal effects.
  - Staggered tag entry for technical skills and coursework.
  - Interactive profile image components with CSS keyframe animations.
  - Dynamic ambient glow effects following cursor movement.
- Responsive Design: Optimized for all device viewports, featuring a dedicated mobile navigation system.
- SEO Standards: Semantic HTML5 structure and optimized meta tags for search engine discovery.

## Technical Stack
- Structure: Semantic HTML5
- Styling: Modern CSS3 (CSS Custom Properties, Flexbox, Grid)
- Scripting: Vanilla JavaScript (ES6+)
- Typography: Inter Font (Google Fonts)

## Directory Structure
- index.html: Core page structure and content.
- style.css: Global styles, layout modules, and responsive breakpoints.
- script.js: Interaction logic and animation controllers.
- profile.jpg: Portfolio hero image assets.

## Local Installation
1. Clone the repository:
   git clone https://github.com/tabassumnabila979-web/My-Portfolio.git
2. Navigate to the directory:
   cd nabila-portfolio
3. Open index.html in a web browser or serve via a local development server.
